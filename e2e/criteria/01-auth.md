# Auth & Account

1. Login screen shows TripJam branding with Sign In / Sign Up tabs
2. User can sign up with a username and password — account is created, lands on home
3. User can sign in with existing username and password — lands on home
4. Username with special characters is sanitized (spaces, symbols stripped)
5. Signing out returns to login screen
6. Session persists across page refresh (user stays logged in)
