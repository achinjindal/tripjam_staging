# Home & Trip Management

1. Home shows "No trips yet" empty state with "Create a Trip" button when user has no trips
2. Home lists all user's trips with name, destination, date range, and day count
3. Each trip card shows a status badge: "Planning" (draft), "Upcoming", "In Progress", or "Past"
4. Clicking a "Planning" (draft) trip opens to Routes page with saved routes
5. Clicking a completed trip opens to Itinerary view
6. Edit button (pencil icon) on a trip card opens to Routes page
7. Delete button removes the trip after confirmation dialog
8. Info tooltip (i icon) shows generated and modified timestamps
9. "New Trip" button appears in header when trips exist
10. User avatar and username show in header with sign-out dropdown
