# Trip Setup Form

1. Setup has multi-step wizard: Destination → Dates → Travelers → Styles → Budget/Pace → Notes
2. Destination field shows autocomplete suggestions from Google Places as user types
3. Multiple destinations can be added as chips (e.g. "Tokyo → Kyoto → Osaka")
4. Destination chips can be removed individually
5. Date picker sets start and end dates; day count is calculated automatically
6. Travelers field accepts a number
7. Style selection allows multiple choices (History, Nature, Food, etc.)
8. Budget has three options: Budget, Mid-range, Luxury
9. Pace has two options: Active, Relaxed
10. Morning start has two options: Early bird, Slow starter
11. Notes field is optional — trip can proceed without notes
12. Arrival/departure details (city, time, mode) can be set optionally
13. Transport mode options: Flight, Train, Bus, Road
14. Form data is preserved when navigating back from Routes page
15. Form data persists in localStorage for refresh survival
